# Amazon-Sales-Report

## Project Objective
To analyze Amazon sales data and build an interactive Power BI dashboard that helps understand sales performance, revenue trends, customer demand, and product-wise insights to support data-driven business decisions. 

## Dataset Used
- <a href="https://github.com/bk9991342-cmyk/Amazon-Sales-Report/blob/main/Amazon%20Sales%20data.xlsx">Dataset</a>

## Process
-  Collected and cleaned Amazon sales data using Excel
-  Loaded and transformed data in Power BI (Power Query)
-  Created calculated measures using DAX
-  Built interactive visuals (KPIs, charts, slicers)
-  Designed a user-friendly dashboard for easy analysis
